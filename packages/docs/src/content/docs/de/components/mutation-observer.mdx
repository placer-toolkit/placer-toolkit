Die Mutation Observer‐Komponente bietet eine schlanke, deklarative Schnittstelle zur [`MutationObserver`‐API](https://developer.mozilla.org/de/docs/Web/API/MutationObserver).

Der Mutation Observer meldet Änderungen an den Inhalten, die er umschließt, über das `pc-mutation`‐Event. Wenn dieses ausgelöst wird, wird eine Sammlung von [`MutationRecord`](https://developer.mozilla.org/de/docs/Web/API/MutationRecord)‐Objekten an `event.detail` angehängt, die Informationen darüber enthalten, wie sich der Inhalt verändert hat.

```demo:open
<div class="mutation-overview">
    <pc-mutation-observer attribute="appearance">
        <pc-button appearance="primary">Klick zum Mutieren</pc-button>
    </pc-mutation-observer>
    <br />
    ⬆️ Klick den Button und schau in die Konsole
</div>

<script>
    const container = document.querySelector(".mutation-overview");
    const mutationObserver = container.querySelector("pc-mutation-observer");
    const button = container.querySelector("pc-button");

    const appearances = ["primary", "success", "neutral", "warning", "danger"];
    let clicks = 0;

    button.addEventListener("click", () => {
        clicks++;
        button.setAttribute(
            "appearance",
            appearances[clicks % appearances.length],
        );
    });

    mutationObserver.addEventListener("pc-mutation", (event) => {
        console.log(event.detail);
    });
</script>

<style>
    .mutation-overview pc-button {
        margin-block-end: var(--pc-spacing-l);
    }
</style>
```

## Demos

### Kindelemente

Verwende das Attribut `child-list`, um auf neue Kindelemente zu achten, die hinzugefügt oder entfernt werden.

```demo
<div class="mutation-child-list">
    <pc-mutation-observer child-list>
        <div class="buttons">
            <pc-button appearance="primary">Button hinzufügen</pc-button>
        </div>
    </pc-mutation-observer>
    ⬆️ Buttons hinzufügen oder entfernen und in die Konsole schauen
</div>

<script>
    const container = document.querySelector(".mutation-child-list");
    const mutationObserver = container.querySelector("pc-mutation-observer");
    const buttons = container.querySelector(".buttons");
    const addButton = container.querySelector('pc-button[appearance="primary"]');

    let index = 0;

    addButton.addEventListener("click", () => {
        const button = document.createElement("pc-button");
        button.textContent = index++;
        buttons.append(button);
    });

    buttons.addEventListener("click", (event) => {
        event.stopPropagation();

        const target = event.target.closest('pc-button:not([appearance="primary"])');

        if (target) {
            target.remove();
        }
    })
</script>

<style>
    .mutation-child-list .buttons {
        display: flex;
        margin-block-end: var(--pc-spacing-l);
        gap: var(--pc-spacing-xs);
        flex-wrap: wrap;
    }
</style>
```
