Placer Toolkit includes classes to set an element’s `border-radius` property. They can be used alongside Placer Toolkit layout utilities, like [Frame](/layout/frame), to round all corners of an element.

Each class corresponds to one of the [`--pc-border-radius-*`](/design-tokens/borders/#border-radius) tokens in your theme.

| Class name                | `border-radius` value       | Preview                                                                                                                                                                                                            |
| ------------------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `pc-border-radius-s`      | `--pc-border-radius-s`      | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-s)"></div>                           |
| `pc-border-radius-m`      | `--pc-border-radius-m`      | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-m)"></div>                           |
| `pc-border-radius-l`      | `--pc-border-radius-l`      | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-l)"></div>                           |
| `pc-border-radius-xl`     | `--pc-border-radius-xl`     | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-xl)"></div>                          |
| `pc-border-radius-square` | `--pc-border-radius-square` | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-square)"></div>                      |
| `pc-border-radius-pill`   | `--pc-border-radius-pill`   | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-pill)"></div>                        |
| `pc-border-radius-circle` | `--pc-border-radius-circle` | <div class="color-palette-swatch" style="border: var(--pc-border-width-s) var(--pc-border-style) var(--pc-color-neutral-border-normal); border-radius: var(--pc-border-radius-circle); aspect-ratio: 1 / 1"></div> |
