Placer Toolkit provides optional native styles, designed to make native HTML elements that aren’t implemented as a web component look good.

## Demos

### Headings

Semantic heading elements with proper hierarchy and styling.

```demo
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

### Paragraphs

Standard paragraph text with optimal spacing and readability.

```demo
<p>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aenean a eros
    tristique, porttitor odio a, efficitur justo. Ut ut mauris vehicula,
    lacinia justo eu, pulvinar est. Duis lobortis purus pellentesque placerat
    rhoncus. Suspendisse varius justo quis sapien faucibus imperdiet. Donec
    id lorem non tortor interdum mattis.
</p>

<p>
    Donec ornare lorem eu sem porta, nec fringilla arcu posuere. Suspendisse
    potenti. Nullam in urna eu diam sagittis cursus. Donec placerat a libero
    quis porta. Vestibulum sollicitudin sed eros non pulvinar. Integer laoreet
    libero mi, id rhoncus dui fringilla quis.
</p>
```

### Blockquotes

Styled quotations that stand out from regular text.

```demo
<blockquote>
    “The web is more of a social creation than a technical one. I designed it for a social effect—to help people work together—and not as a technical toy.”
    <footer>— <cite>Tim Berners‐Lee</cite></footer>
</blockquote>
```

```demo

    <p><strong>Bold</strong></p>
    <p><em>Italic</em></p>
    <p><u>Underline</u></p>
    <p><s>Strikethrough</s></p>
    <p><del>Deleted</del></p>
    <p><ins>Inserted</ins></p>
    <p><small>Small</small></p>
    <p><span>Subscript <sub>Sub</sub></span></p>
    <p><span>Superscript <sup>Sup</sup></span></p>
    <p><abbr title="Abbreviation">Abbr.</abbr></p>
    <p><mark>Highlighted</mark></p>
    <p><a href="#">Link text</a></p>
    <p><code>Inline code</code></p>
    <p><kbd>Keyboard</kbd></p>

<style>
    .native-inline {
        columns: 2;
        gap: var(--pc-spacing-l);
    }
```

## Importing

If you imported `placer.css`, you can skip this section. But if you cherry picked from all the CSS files, you can use these snippets to import these native styles.

<pc-tab-group>
    <pc-tab slot="navigation" panel="in-html">In HTML</pc-tab>
    <pc-tab slot="navigation" panel="in-css">In CSS</pc-tab>

    <pc-tab-panel name="in-html">
        To import native styles from the CDN with a link tag, copy this snippet and paste it in your HTML:

        ```html
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/placer-toolkit@1.0.0-alpha.1/cdn/styles/native.css" />
        ```
    </pc-tab-panel>
    <pc-tab-panel name="in-css">
        To import native styles with an `@import` at‐rule, copy this snippet and paste it in your global CSS file:

        ```css
        @import url("https://cdn.jsdelivr.net/npm/placer-toolkit@1.0.0-alpha.1/cdn/styles/native.css");
        ```
    </pc-tab-panel>
</pc-tab-group>
