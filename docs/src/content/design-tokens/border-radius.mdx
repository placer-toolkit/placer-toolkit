---
title: Border radius
---

Border radii are used to give shapes a soft, rounded and more friendly feel. They use rem units so they scale with the base font size. The px values are based on a root of 16px.

| Design token                | Default value  |                                                 Example                                                 |
| --------------------------- | -------------- | :-----------------------------------------------------------------------------------------------------: |
| `--pc-border-radius-s`      | 0.25rem (4px)  |         <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-s)"></div>         |
| `--pc-border-radius-m`      | 0.25rem (8px)  |         <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-m)"></div>         |
| `--pc-border-radius-l`      | 0.25rem (12px) |         <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-l)"></div>         |
| `--pc-border-radius-xl`     | 0.25rem (16px) |        <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-xl)"></div>         |
| `--pc-border-radius-circle` | 50%            |      <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-circle)"></div>       |
| `--pc-border-radius-pill`   | 10000px        | <div class="border-radius-demo" style="border-radius: var(--pc-border-radius-pill); width: 7rem"></div> |
