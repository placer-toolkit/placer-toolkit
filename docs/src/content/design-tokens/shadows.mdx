Shadows are used to give elements the appearance of being raised off the page. Use them with the `box-shadow` property. These are especially useful for menus, popovers and dialogs.

| Token            |                                 Example                                 |
| ---------------- | :---------------------------------------------------------------------: |
| `--pc-shadow-xs` | <div class="shadow-demo" style="box-shadow: var(--pc-shadow-xs)"></div> |
| `--pc-shadow-s`  | <div class="shadow-demo" style="box-shadow: var(--pc-shadow-s)"></div>  |
| `--pc-shadow-m`  | <div class="shadow-demo" style="box-shadow: var(--pc-shadow-m)"></div>  |
| `--pc-shadow-l`  | <div class="shadow-demo" style="box-shadow: var(--pc-shadow-l)"></div>  |
| `--pc-shadow-xl` | <div class="shadow-demo" style="box-shadow: var(--pc-shadow-xl)"></div> |
