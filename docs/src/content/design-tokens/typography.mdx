Typography is used to maintain a consistent set of font styles throughout your app.

## Font family

The default font stack is simple and elegant, and has a lot of fallbacks if the main font isn’t available.

| Design token      | Default value                                                                                                                                                                                               | Example                                                                                      |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| `--pc-font-sans`  | `"Quicksand", system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI Variable", "Segoe UI", "Roboto", "Open Sans", "Helvetica Neue", sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol"` | <span style="font-family: var(--pc-font-sans)">Sphinx of black quartz, judge my vow.</span>  |
| `--pc-font-serif` | `"Playfair Display", "Georgia", "Times New Roman", serif`                                                                                                                                                   | <span style="font-family: var(--pc-font-serif)">Sphinx of black quartz, judge my vow.</span> |
| `--pc-font-mono`  | `"JetBrains Mono", "Source Code Pro", "IBM Plex Mono", "Consolas", "Menlo", monospace`                                                                                                                      | <span style="font-family: var(--pc-font-mono)">Sphinx of black quartz, judge my vow.</span>  |

## Font size

Font sizes use `rem` units so they scale with the base font size. The `px` values displayed are based on a 16px root font size.

| `--pc-font-size-m`     | 1rem (16px)     | <span style="font-size: var(--pc-font-size-m)">Aa</span>     |

## Font weights

Font weights range from 300 (Light) to 700 (Bold), letting you fine‐tune emphasis. Our default variable fonts support any value in this range, while other non‐variable fallbacks may use standard weight tokens.

| Design token                | Default value | Example                                                                                                |
| --------------------------- | ------------- | ------------------------------------------------------------------------------------------------------ |
| `--pc-font-weight-light`    | 300           | <span style="font-weight: var(--pc-font-weight-light)">Sphinx of black quartz, judge my vow.</span>    |
| `--pc-font-weight-normal`   | 400           | <span style="font-weight: var(--pc-font-weight-normal)">Sphinx of black quartz, judge my vow.</span>   |
| `--pc-font-weight-medium`   | 500           | <span style="font-weight: var(--pc-font-weight-medium)">Sphinx of black quartz, judge my vow.</span>   |
| `--pc-font-weight-semibold` | 600           | <span style="font-weight: var(--pc-font-weight-semibold)">Sphinx of black quartz, judge my vow.</span> |
| `--pc-font-weight-bold`     | 700           | <span style="font-weight: var(--pc-font-weight-bold)">Sphinx of black quartz, judge my vow.</span>     |
