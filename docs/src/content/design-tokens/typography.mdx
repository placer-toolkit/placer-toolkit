## Font size

Our font size system uses the major second ratio (<math display="inline"><mrow><mtext>r</mtext><mo>=</mo><mn>1,125</mn></mrow></math>): smaller sizes scale down linearly by <math display="inline"><mi>r</mi></math>, while larger sizes scale up by the squared ratio (<math display="inline"><msup><mi>r</mi><mn>2</mn></msup></math>) to maximise visual impact.

Each value uses `rem` units and is rounded to the nearest whole pixel when rendered with [`round()`](https://developer.mozilla.org/en-US/docs/Web/CSS/round).

You can use `--pc-font-size-scale` to increase or decrease all font sizes at once. By default, this multiplier is 1.

| Custom property        | Default value                                                        | Example                                                        |
| ---------------------- | -------------------------------------------------------------------- | -------------------------------------------------------------- |
| `--pc-font-size-xxs`   | `round(calc(var(--pc-font-size-xs) / 1.125), 1px)` (11 px)           | <span style="font-size: var(--pc-font-size-xxs)">AaBb</span>   |
| `--pc-font-size-xs`    | `round(calc(var(--pc-font-size-s) / 1.125), 1px)` (12 px)            | <span style="font-size: var(--pc-font-size-xs)">AaBb</span>    |
| `--pc-font-size-s`     | `round(calc(var(--pc-font-size-m) / 1.125), 1px)` (14 px)            | <span style="font-size: var(--pc-font-size-s)">AaBb</span>     |
| `--pc-font-size-m`     | `round(calc(1rem * var(--pc-font-size-scale)), 1px)` (16 px)         | <span style="font-size: var(--pc-font-size-m)">AaBb</span>     |
| `--pc-font-size-l`     | `round(calc(var(--pc-font-size-m) * 1.125 * 1.125), 1px)` (20 px)    | <span style="font-size: var(--pc-font-size-l)">AaBb</span>     |
| `--pc-font-size-xl`    | `round(calc(var(--pc-font-size-l) * 1.125 * 1.125), 1px)` (25 px)    | <span style="font-size: var(--pc-font-size-xl)">AaBb</span>    |
| `--pc-font-size-xxl`   | `round(calc(var(--pc-font-size-xl) * 1.125 * 1.125), 1px)` (32 px)   | <span style="font-size: var(--pc-font-size-xxl)">AaBb</span>   |
| `--pc-font-size-xxxl`  | `round(calc(var(--pc-font-size-xxl) * 1.125 * 1.125), 1px)` (41 px)  | <span style="font-size: var(--pc-font-size-xxxl)">AaBb</span>  |
| `--pc-font-size-xxxxl` | `round(calc(var(--pc-font-size-xxxl) * 1.125 * 1.125), 1px)` (52 px) | <span style="font-size: var(--pc-font-size-xxxxl)">AaBb</span> |

You can also use these two custom properties to make any font size proportionately smaller or larger to its parent.

| Custom property          | Default value                           |
| ------------------------ | --------------------------------------- |
| `--pc-font-size-smaller` | `round(calc(1em / 1.125), 1px)`         |
| `--pc-font-size-larger`  | `round(calc(1em * 1.125 * 1.125), 1px)` |

## Font weights

Font weights range from 300 (Light) to 700 (Bold), letting you fine‐tune emphasis. Our default variable fonts support any value in this range, while other non‐variable fallbacks may use standard weight tokens.

| Custom property             | Default value | Example                                                                                                |
| --------------------------- | ------------- | ------------------------------------------------------------------------------------------------------ |
| `--pc-font-weight-light`    | `300`         | <span style="font-weight: var(--pc-font-weight-light)">Sphinx of black quartz, judge my vow.</span>    |
| `--pc-font-weight-normal`   | `400`         | <span style="font-weight: var(--pc-font-weight-normal)">Sphinx of black quartz, judge my vow.</span>   |
| `--pc-font-weight-medium`   | `500`         | <span style="font-weight: var(--pc-font-weight-medium)">Sphinx of black quartz, judge my vow.</span>   |
| `--pc-font-weight-semibold` | `600`         | <span style="font-weight: var(--pc-font-weight-semibold)">Sphinx of black quartz, judge my vow.</span> |
| `--pc-font-weight-bold`     | `700`         | <span style="font-weight: var(--pc-font-weight-bold)">Sphinx of black quartz, judge my vow.</span>     |

## Line height

Line heights control the distance between lines of text and are unitless to scale proportionately with the font size. For readability, `--pc-line-height-normal`, recommended for paragraphs, should be `1.6` or greater.

## Links

Together with [`--pc-color-link`](/design-tokens/colour/#text), these tokens add text decoration to `<a>` elements to signal their role as hyperlinks.

| Custom property                | Default value                   |
| ------------------------------ | ------------------------------- |
| `--pc-link-decoration-default` | `underline dotted currentColor` |
| `--pc-link-decoration-hover`   | `underline solid currentColor`  |
| `--pc-link-decoration-active`  | `underline solid currentColor`  |
