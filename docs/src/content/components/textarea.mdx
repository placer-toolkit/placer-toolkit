Textareas collect data from the user and allow multiple lines of text.

```demo:open
<pc-textarea></pc-textarea>
```

<pc-callout appearance="primary">
    <pc-icon
        library="default"
        icon-style="solid"
        name="book-open"
        slot="icon"
    ></pc-icon>
    This component works well with standard `<form>` elements. Please refer to the [form controls](/get-started/form-controls) page to learn more about form submission and client‐side validation.
</pc-callout>

## Demos

### Labels

Use the `label` attribute to give the textarea an accessible label. For labels that contain HTML, use the `label` slot instead.

```demo
<pc-textarea label="Comments"></pc-textarea>
```

### Hints

Add a descriptive hint to a textarea with the `hint` attribute. For hints that contain HTML, use the `hint` slot instead.

```demo
<pc-textarea label="Feedback" hint="Please tell us what you think."></pc-textarea>
```

### Rows

Use the `rows` attribute to change the number of text rows that get shown.

```demo
<pc-textarea rows="2"></pc-textarea>
```

### Placeholders

Use the `placeholder` attribute to add a placeholder.

```demo
<pc-textarea placeholder="Type something…"></pc-textarea>
```

### Filled

Add the `filled` attribute to draw a filled textarea.

```demo
<pc-textarea placeholder="Type something…" filled></pc-textarea>
```

### Disabled

Use the `disabled` attribute to disable the textarea.

```demo
<pc-textarea placeholder="Type something…" disabled></pc-textarea>
```

### Sizes

Use the `size` attribute to change the textarea’s size.

```demo
<pc-textarea size="small" placeholder="Small"></pc-textarea>
<br />
<pc-textarea size="medium" placeholder="Medium"></pc-textarea>
<br />
<pc-textarea size="large" placeholder="Large"></pc-textarea>
```

### Prevent resizing

By default, textareas can be resized vertically by the user. To prevent resizing, set the `resize` attribute to `none`.

```demo
<pc-textarea resize="none"></pc-textarea>
```

### Expand with content

Textareas will automatically resize to fit their content when `resize` is set to `auto`.

```demo
<pc-textarea resize="auto"></pc-textarea>
```
