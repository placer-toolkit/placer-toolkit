## Demos

### Basic card

Basic cards aren’t very exciting, but they can display any content you want them to.

<CodeDemo>
{`<pc-card class="card-basic">
    This is just a basic card. No image, no header and no footer. Just your content.
</pc-card>

<style>
    .card-basic {
        max-width: 300px;
    }
</style>`}
</CodeDemo>

### With header

Headers can be used to display titles and more.

<CodeDemo>
{`<pc-card class="card-header">
    <header slot="header">
        <strong>Header title</strong>
        <pc-icon-button
            library="default"
            icon-style="solid"
            name="gear"
            label="Settings"
        ></pc-icon-button>
    </header>

    This card has a header. You can put all sorts of things in it!
</pc-card>

<style>
    .card-header {
        max-width: 300px;
    }

    .card-header header {
        display: flex;
        align-items: center;
        justify-content: space-between;
    }
</style>`}
</CodeDemo>

### With footer

Footers can be used to display actions, summaries or other relevant content.

<CodeDemo>
{`<pc-card class="card-footer">
    This card has a footer. You can put all sorts of things in it!

    <footer slot="footer">
        <pc-button appearance="primary" pill>
            <pc-icon
                library="default"
                icon-style="solid"
                name="cart-shopping"
                slot="prefix"
            ></pc-icon>
            Buy
        </pc-button>
        <pc-button appearance="text" pill>
            More info
            <span class="pc-visually-hidden">about this card</span>
        </pc-button>
    </footer>
</pc-card>

<style>
    .card-footer {
        max-width: 300px;
    }

    .card-footer footer {
        display: flex;
        align-items: center;
        justify-content: flex-end;
        gap: var(--pc-spacing-xs);
    }
</style>`}
</CodeDemo>

### Images

Cards accept an `image` slot. The image is displayed at the top of the card and stretches to fit.

<CodeDemo>
{`<pc-card class="card-image">
    <img
        slot="image"
        src="https://placehold.co/300x200"
        alt="A placeholder image with grey background and dark grey text in resolution 300 by 200 pixels"
    />

    This is a placeholder image with a grey background and dark grey text with a 300×200 intrinsic resolution.
</pc-card>

<style>
    .card-image {
        max-width: 300px;
    }
</style>`}
</CodeDemo>
