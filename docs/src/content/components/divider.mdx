Dividers are used to visually separate or group elements.

<CodeDemo open>
{`<pc-divider></pc-divider>`}
</CodeDemo>

## Demos

### Stroke width

Use the `--stroke-width` custom property to change the stroke width of the divider.

<CodeDemo>
{`<pc-divider style="--stroke-width: 4px"></pc-divider>`}
</CodeDemo>

### Colour

Use the `--color` custom property to change the colour of the divider.

<CodeDemo>
{`<pc-divider style="--color: var(--pc-color-primary-500)"></pc-divider>`}
</CodeDemo>

### Spacing

Use the `--spacing` custom property to change the amount of space between the divider and its neighbouring elements.

<CodeDemo>
{`<div style="text-align: center">
    Above
    <pc-divider style="--spacing: 2rem"></pc-divider>
    Below
</div>`}
</CodeDemo>

### Vertical

Add the `vertical` attribute to draw the divider in a vertical orientation. The divider will span the full height of its container. Vertical dividers work especially well inside of a flex container.

<CodeDemo>
{`<div style="display: flex; align-items: center; height: 2rem">
    First
    <pc-divider vertical></pc-divider>
    Second
    <pc-divider vertical></pc-divider>
    Last
</div>`}
</CodeDemo>

### Menu dividers

Use dividers in [menus](/components/menu) to visually group menu items.

<CodeDemo>
{`<pc-menu style="max-width: 200px">
    <pc-menu-item value="1">Option 1</pc-menu-item>
    <pc-menu-item value="2">Option 2</pc-menu-item>
    <pc-menu-item value="3">Option 3</pc-menu-item>
    <pc-divider></pc-divider>
    <pc-menu-item value="4">Option 4</pc-menu-item>
    <pc-menu-item value="5">Option 5</pc-menu-item>
    <pc-menu-item value="6">Option 6</pc-menu-item>
</pc-menu>`}
</CodeDemo>
