Tooltips display additional information based on a specific action.

A tooltip’s target is its _first child element_, so you should only wrap one element inside the tooltip. If you need the tooltip to show up for multiple elements, nest them inside a container first.

```demo
<pc-tooltip content="This is a tooltip!">
    <pc-button>Hover me</pc-button>
</pc-tooltip>
```

## Demos

### Click trigger

Set the `trigger` attribute to `click` to toggle the tooltip on click instead of hover.

```demo
<pc-tooltip content="Click again to dismiss" trigger="click">
    <pc-button>Click to toggle</pc-button>
</pc-tooltip>
```

### Manual trigger

Tooltips can be controlled programmatically by setting the `trigger` attribute to `manual`. Use the `open` attribute to control when the tooltip is shown.

```demo
<pc-button style="margin-inline-end: 4rem">Toggle manually</pc-button>

<pc-tooltip class="tooltip-manual" content="This is an avatar" trigger="manual">
    <pc-avatar label="User"></pc-avatar>
</pc-tooltip>

<script>
    const tooltip = document.querySelector(".tooltip-manual");
    const toggle = tooltip.previousElementSibling;

    toggle.addEventListener("click", () => (tooltip.open = !tooltip.open));
</script>
```

### Removing arrows

You can control the size of tooltip arrows by overriding the `--pc-tooltip-arrow-size` custom property. To remove them, set the value to `0` as shown below.

```demo
<pc-tooltip
    content="This is a tooltip without an arrow."
    style="--pc-tooltip-arrow-size: 0"
>
    <pc-button>No arrow</pc-button>
</pc-tooltip>
```

To override it globally, set it in a root block in your styles after the Placer Toolkit styles are loaded.

```css
:root {
    --pc-tooltip-arrow-size: 0;
}
```

### HTML in tooltips

Use the `content` slot to create tooltips with HTML content. Tooltips are designed only for text and presentational elements. Avoid placing interactive content, such as buttons, links and form controls, in a tooltip.

```demo
<pc-tooltip>
    <div slot="content">
        I’m not <strong>just</strong> a tooltip, I’m a <em>tooltip</em> with HTML!
    </div>

    <pc-button>Hover me</pc-button>
</pc-tooltip>
```

### Set a maximum width

Use the `--max-width` custom property to change the width the tooltip can grow to before it wraps.

```demo
<pc-tooltip
    content="This tooltip will wrap after 80 pixels."
    style="--max-width: 80px"
>
    <pc-button>Hover me</pc-button>
</pc-tooltip>
```
