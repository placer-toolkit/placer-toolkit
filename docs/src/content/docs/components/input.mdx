Inputs collect data from the user.

```demo:open
<pc-input></pc-input>
```

<pc-callout appearance="primary">
    <pc-icon
        library="default"
        icon-style="solid"

        slot="icon"

    This component works well with standard `<form>` elements. Please refer to the [form controls](/docs/get-started/form-controls) page to learn more about form submission and client‐side validation.
</pc-callout>

## Demos

### Labels

Use the `label` attribute to give the input an accessible label. For labels that contain HTML, use the `label` slot instead.

```demo
<pc-input label="What’s your name?"></pc-input>
```

### Hints

Add a descriptive hint to an input with the `hint` attribute. For hints that contain HTML, use the `hint` slot instead.

```demo
<pc-input
    label="Nickname"
    hint="What would you like people to call you?"
></pc-input>
```

### Placeholders

Use the `placeholder` attribute to add a placeholder.

```demo
<pc-input placeholder="Type something…"></pc-input>
```

### Clearable

Add the `clearable` attribute to add a clear button when the input has content.

```demo
<pc-input placeholder="Clearable" clearable></pc-input>
```

### Password toggle

Add the `password-toggle` attribute to add a toggle button that will show the password when activated.

```demo
<pc-input type="password" placeholder="Password" password-toggle></pc-input>
```

### Filled

Add the `filled` attribute to draw a filled input.

```demo
<pc-input placeholder="Type something…" filled></pc-input>
```

### Disabled

Use the `disabled` attribute to disable the input.

```demo
<pc-input placeholder="Disabled" disabled></pc-input>
```

### Size

Use the `size` attribute to change the input’s size.

```demo
<pc-input size="small" placeholder="Small"></pc-input>

<br />

<pc-input size="medium" placeholder="Medium"></pc-input>

<br />

<pc-input size="large" placeholder="Large"></pc-input>
```

### Pill

Use the `pill` attribute to give inputs a pill‐shaped look.

```demo
<pc-input size="small" placeholder="Small" pill></pc-input>

<br />

<pc-input size="medium" placeholder="Medium" pill></pc-input>

<br />

<pc-input size="large" placeholder="Large" pill></pc-input>
```

### Input types

Use the `type` attribute to control the input type that the browser renders.

```demo
<pc-input type="email" placeholder="E‐mail"></pc-input>

<br />

<pc-input type="number" placeholder="Number"></pc-input>

<br />

<pc-input type="date" placeholder="Date"></pc-input>
```

### Prefixes and suffixes

Use the `prefix` and `suffix` slots to add presentational icons and text. Avoid slotting in interactive elements, such as buttons and links.

```demo
<pc-input size="small" placeholder="Small">
    <pc-icon
        library="default"
        icon-style="solid"
        name="house"
        slot="prefix"
    ></pc-icon>
    <pc-icon
        library="default"
        icon-style="solid"
        name="message"
        slot="suffix"
    ></pc-icon>
</pc-input>

<br />

<pc-input size="medium" placeholder="Medium">
    <pc-icon
        library="default"

    ></pc-icon>
```
