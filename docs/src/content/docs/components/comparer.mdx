Compare visual differences between similar content with a sliding panel.

For best results, use images that share the same dimensions. The slider can be controlled by dragging or pressing the <kbd aria-label="Left">←</kbd> and <kbd aria-label="Right">→</kbd> arrow keys.

<pc-callout appearance="primary">
    <pc-icon
        library="default"
        icon-style="solid"
        name="keyboard"
        slot="icon"
    ></pc-icon>
    Press <AutoKeyboardSwitch keyName="shift" /> + <kbd aria-label="Left">←</kbd> or <kbd aria-label="Right">→</kbd> arrow key to move the slider in intervals of 10, or <kbd>Home</kbd> and <kbd>End</kbd> to jump to the beginning or the end.
</pc-callout>

## Demos

Use the `position` attribute to set the initial position of the slider. This is a percentage from `0` to `100`.
